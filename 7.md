# Задача. Поиск максимальной согласованной группы
Пользователь вводит описание людей и языков, 
которыми они владеют, по одной записи на строку. 
Ввод заканчивается строкой <b>END</b>.
Найти наибольшую группу людей, 
которые имеют хотя бы один общий язык, и вывести 
этот общий язык и список людей в группе (отсортированный по имени).
Если таких языков несколько — 
выбрать лексикографически минимальный язык.

| Ввод                                                                                    | Вывод                     |
|-----------------------------------------------------------------------------------------|---------------------------|
| Ivan: Python Java<br>Anna: Python C++<br>Boris: Java C++<br>Olga: Python<br><b>END</b>  | Python<br>Anna Ivan Olga  |
