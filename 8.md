# Задача. Нормализация последовательностей
Напишите функции, которые будут:
- принимать строку чисел, разделённых запятыми, 
а возвращать список целых чисел

| Принимает       | Возвращает               |
|-----------------|--------------------------|
| 1,2,3,2,3,5,1,4 | [1, 2, 3, 2, 3, 5, 1, 4] |

- принимать в качестве аргумента список целых чисел, 
разделять его на список списков, 
внутри каждого из которых неубывающая последовательность 

| Принимает                | Возвращает                     |
|--------------------------|--------------------------------|
| [1, 2, 3, 2, 3, 5, 1, 4] | [[1, 2, 3], [2, 3, 5], [1, 4]] |

- принимать список целых чисел 
и производить на них Z-нормализацию
(из каждого числа вычитается среднее арифметическое, 
результат делится на стандартное оклонение)

| Принимает | Возвращает       |
|-----------|------------------|
| [1,2,3]   | [-1.0, 0.0, 1.0] |

- принимать список списков чисел, 
возвращать список Z-нормализованных списков

| Принимает                      | Возвращает                                                                                                                     |
|--------------------------------|--------------------------------------------------------------------------------------------------------------------------------|
| [[1, 2, 3], [2, 3, 5], [1, 4]] | [[-1.0, 0.0, 1.0], [-0.8728715609439696, -0.21821789023599247, 1.0910894511799618], [-0.7071067811865476, 0.7071067811865476]] |

- оркестрировать предущими функциями, обрабатывая вводимые строки до ввода **END**
